# 🏨 Sistema de Hospedagem

Projeto simples em C# utilizando .NET para simular um sistema básico de hospedagem.

## 📌 Funcionalidades

* Cadastro de hóspedes
* Criação de suítes
* Realização de reservas
* Cálculo de valor da diária
* Exibição da quantidade de hóspedes

## 🚀 Tecnologias

* .NET
* C#

## ▶️ Como executar

```bash
dotnet run
```

## 💡 Exemplo

O sistema cria hóspedes, associa a uma suíte e realiza uma reserva, exibindo:

* Quantidade de hóspedes
* Valor total da estadia

Projeto com foco em prática de conceitos básicos de programação orientada a objetos.
